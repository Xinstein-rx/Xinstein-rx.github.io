1149 Dangerous Goods Packaging (25分)

2020-06-01

When shipping goods with containers, we have to be careful not to pack some incompatible goods into the same container, or we might get ourselves in serious trouble. For example, oxidizing agent （氧化剂） must not be packed with flammable liquid （易燃液体）, or it can cause explosion.

Now you are given a long list of incompatible goods, and several lists of goods to be shipped. You are supposed to tell if all the goods in a list can be packed into the same container.

Input Specification:

Each input file contains one test case. For each case, the first line gives two positive integers: N (≤104), the number of pairs of incompatible goods, and M (≤100), the number of lists of goods to be shipped.

Then two blocks follow. The first block contains N pairs of incompatible goods, each pair occupies a line; and the second one contains M lists of goods to be shipped, each list occupies a line in the following format:

K G[1] G[2] ... G[K]

where K (≤1,000) is the number of goods and G[i]‘s are the IDs of the goods. To make it simple, each good is represented by a 5-digit ID number. All the numbers in a line are separated by spaces.

Output Specification:

For each shipping list, print in a line Yes if there are no incompatible goods in the list, or No if not.

解题思路

终于有了一道自己做出来的题，刚开始没做对，是因为想着就只有一对物品不可以在一起，没想到物品还可以和多个物品不能放一起，改完了之后就对了。题解地方法也差不多，用的是map<int, vector > ,把不能相容的物品都放在向量里，然后输入装入的物品，把他们都标记为1，再继续检查每个物品不能相容的物品标记是不是1，是1就不相融，这个我感觉要比那个查找简单点。然后还通过这个题学了下find函数，一开始不对是因为忘了加algorithm头文件。还学到了关于string的一些查找函数。比如find函数返回的是一个int值，就是匹配的字符串第一个字符所在位置，是一个全匹配，只要有一个不符合就不行，还有比如s.find_first_of(s2),返回s2中某个字符第一次出现的位置，他不是全匹配，只要有一个字符符合就行，s1 = “abcdef”,s2 = “dek”,s1.find_first_of(s2) = 3,而find_last_of()是从后面往前找，还是这个例子，s1.find_last_of(s2) = 4。
