Now you are asked to find the minimum number of swaps need to sort the given permutation of the first N nonnegative integers.

Input Specification:

Each input file contains one test case, which gives a positive N (≤105) followed by a permutation sequence of {0, 1, …, N−1}. All the numbers in a line are separated by a space.

解题思路

这个题我觉得是个思维题叭，反正当时看了半天也不会，还想着是不是用什么逆序数的方法解决，但看完题解之后发现并不是，题解也是看了很久，看完之后还是并不明白，让我自己想的话我觉得我是额。。。根本想不出来的，等过段时间再回来复习一下，加深下印象。这个题就是用一个数组保存数的位置，关键我觉得还是在于理解，位置的大小就是该位置数的大小，用0作为哨兵，把他所在的位置与该位置应该有的数字的位置交换，即swap(a[0], a[a[0]),直到他换到他应该有的位置，这样就没法再换了，再判断他原来的位置是不是放好了数，如果放好了不用管，继续判断，如果没有，就让他和a[0]交换（待会儿还会把它交换回去的），这样遍历一遍后，数组就有序了，emmmm，现在说完一遍才刚刚有点明白，真是让我在想一个月也想不明白啊呜呜呜。
