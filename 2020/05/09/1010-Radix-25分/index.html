1010 Radix (25分)

2020-05-09

Given a pair of positive integers, for example, 6 and 110, can this equation 6 = 110 be true? The answer is yes, if 6 is a decimal number and 110 is a binary number.

Now for any pair of positive integers *N1*and N2, your task is to find the radix of one number while that of the other is given.

Input Specification:

Each input file contains one test case. Each case occupies a line which contains 4 positive integers:

N1 N2 tag radix

Here N1 and N2 each has no more than 10 digits. A digit is less than its radix and is chosen from the set { 0-9, a-z } where 0-9 represent the decimal numbers 0-9, and a-z represent the decimal numbers 10-35. The last number radix is the radix of N1 if tag is 1, or of N2 if tag is 2.

Output Specification:

For each test case, print in one line the radix of the other number so that the equation N1 = N2 is true. If the equation is impossible, print Impossible. If the solution is not unique, output the smallest possible radix.

解题思路

这个题是一个很让俺心痛的一个题（当然还不能说是最心痛，因为以后谁知道还会有多么心痛的题呢），好像是19个测试点，过了18个，就剩一个一分的，真的是十分难受，还没做的时候不小心扫到了题解上的知识点总结，说是二分，有点慌，因为二分并不怎么会用，只知道方法是什么，但是自己做了做，没有用到二分，emmm果真就不对哈哈哈，自己当时的想法是都把他们转化成十进制，然后挨个进制（从字符串字符代表数最大的下一个开始，因为数字再大也不会超了他们的进制）转化另一个数，直到他们想等或者那个数大于给出进制的数，这样就造成了一个问题，就是不知道多大进制停下，我就大胆猜了个1000，然后就过了18个点，我后来改了改换成999，1101，即使是多了一个数就有好几个测试点过不了amazing，我现在也不知道为什么。然后正确的做法是，找到字符串字符代表数的最大值，他的下一个值（也就是他可能的最小的进制）开始，作为low，然后另一个数与low其中那个较大的作为high，因为如果进制超过了另一个数（可以相等），那样无论怎么转化都是大于另一个数，用这两个上下限进行二分，ll t = con(s, mid) if(t < 0 || t > sum) high = mid - 1;else if(t < sum) low = mid + 1;else ans = mid,break; 这里尤其要注意第一个条件中的t<0，这是溢出的判断，因为在转化过程中可能会发生溢出，这样转化的值就变成小于的了（get）。还有在这个题中学到了柳婼小姐姐用的一个函数，max_element() 与 min_element()，然后网上查了查，说是它们返回的都是迭代器，如果要取内容，要在前面加 * ，然后还可以自定义cmp函数，还可以用于结构体，数组，vector，string。


max_element() 与 min_element()用法如下：

  1. 用的时候要 #include<algorithm>头文件，返回的是一个迭代器，像指针一样输出值的话要在前面加 *
  2. 第三个参数 cmp是可选的，max_element() 与 min_element()默认从小到大排列，max_element()输出最后一个值，min_element输出第一个值，但是如果自定义的 cmp函数是从大到小排列的，出书结果就会相反。
  3. 这个函数可用于vector，也可用于array，当容器内是结构体时，就需要用到 cmp函数了
