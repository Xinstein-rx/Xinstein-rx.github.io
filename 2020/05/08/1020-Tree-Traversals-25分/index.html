1020 Tree Traversals (25分)

2020-05-08

Suppose that all the keys in a binary tree are distinct positive integers. Given the postorder and inorder traversal sequences, you are supposed to output the level order traversal sequence of the corresponding binary tree.

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer N (≤30), the total number of nodes in the binary tree. The second line gives the postorder sequence and the third line gives the inorder sequence. All the numbers in a line are separated by a space.

Output Specification:

For each test case, print in one line the level order traversal sequence of the corresponding binary tree. All the numbers in a line must be separated by exactly one space, and there must be no extra space at the end of the line.

解题思路

这个题是2018年暑假留校的时候学长给我们讲的一个例题，然后我们数据结构课堂上也学了相关的内容，每次都是当时会了，但后来又忘emmm是脑子不好使呢，还是当时学的时候真就没认真学呢，感觉也挺认真的啊，然后二叉树的层序遍历都差点不会写了，这个题常规方法就是基于后序和中序遍历构造二叉树，然后再对二叉树进行层序遍历。题解给了个不用构建二叉树的方法，因为通常方法中的那个build函数，就是一种先序构造，所以在构造的时候把结构体（节点编号，节点值， 相当于节点）放入一个vector里，最后在对vector进行以index为关键字的排序就得到了层序遍历结果，如果是知道后序中序求前序，连vector和结构体都不用，直接构造过程中输出根节点值就好了，因为构造就是通过先序构造的~还学到一种新的把结构体放入vector里面的方法，直接放入ans.push_back( {index, data} ).不过好像是c++11里面的内容。
