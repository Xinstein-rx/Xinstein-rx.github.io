1093 Count PAT's (25分)

2020-05-25

The string APPAPT contains two PAT‘s as substrings. The first one is formed by the 2nd, the 4th, and the 6th characters, and the second one is formed by the 3rd, the 4th, and the 6th characters.

Now given any string, you are supposed to tell the number of PAT‘s contained in the string.

Input Specification:

Each input file contains one test case. For each case, there is only one line giving a string of no more than 105 characters containing only P, A, or T.

Output Specification:

For each test case, print in one line the number of PAT‘s contained in the string. Since the result may be a huge number, you only have to output the result moded by 1000000007.

这个题乙级里面有，但是还是不会做，hhhhh，不出十五分钟就看了题解emmmm，题解的思路是定位好A，则这个A前面P的个数*A后面T的个数，就是这个A可以组成的PAT的个数，虽然看完题解知道了，哦原来是这么回事，但是自己写还是不太会写，题解给出的写法是，先计数好T的个数，然后从前往后遍历，遇到T就–，遇到P就++，然后遇到A就相乘，这个题解把它归到了逻辑题里面，可能也是自己想的不够，一般这种思维题都不太会做。我还想到把字符串逆过来，定位T，看后面有多少A和P，然后乘起来，后来想了想不太行，这个题也为我们提供了一种思路，就是三个变量都在变，不如就定位中间那个，看两边，如果都不行，再换另外的试试。
